Welcome to BG-Omics Thermotoga-project repository.

This project aims to be able to compare protein sequences of one species from Thermotoga organism to another species in the same genus.The desired result is a data table containing all locus tags of matched protein from the given organisms.

Technology

We are using different technologies to achieve the goal.Here are the main technologies we are using:
1.Python(BioPython).
2.MySql.
3.PHP.
4.Javascript.
